# Private Marketing Agency Blueprint

**Build your own AI-powered marketing team inside Claude Code. No coding required. About 2.5 hours of focused work.**

---

## The problem we're solving

You spend hours every week on marketing. Writing social posts, designing graphics, planning what to talk about next week, researching what your competitors are doing, making videos, updating your website.

Every one of those tasks is the same shape. Repetitive. Structured. Volume-dependent. You know what good looks like, but the doing takes forever. You know AI can help, but the tools feel disconnected. One app for writing, another for images, another for video, another for scheduling. Nothing talks to each other.

In the next 2.5 hours you build a system where all of it works together. You type one command and get a week of content: researched, planned, written in your brand voice, designed, and ready to publish.

---

## How to spot what's worth automating

Most people pick the wrong process to automate first. They reach for something complicated and judgement-heavy, then spend a week building it and end up with a tool that is worse than doing it manually.

The trick is the opposite. Look for processes that are **repetitive AND mostly production rather than judgement**. That is the sweet spot.

Marketing content is the textbook case. The process repeats every week. The output is structured (posts, visuals, videos, calendar). The judgement part (is this the right message for my audience?) is small compared to the production part (writing, designing, formatting, scheduling). So it is a perfect automation candidate.

7. **Landing Page** that builds and deploys a page for your business

You type one command. The pipeline runs. You review the output and publish.

## How to start

This course is interactive. You do not read it. You open it in Claude Code and let Claude walk you through it.

### Step 1: Open this folder in Claude Code

If you have the Claude Code desktop app, choose "Open Folder" and select the Private Marketing Agency Blueprint folder you downloaded.

If you are using Claude Code from the terminal, open a terminal in this folder and run `claude`.

### Step 2: Type this

In the chat input, type:

```
Start Lesson 1
```

Claude will read the lesson file, become your instructor for the next twenty minutes, and walk you through the build step by step.

When you finish a lesson, type `Start Lesson 2`, then `Start Lesson 3`, and so on. Each lesson builds directly on the last. Do not skip ahead.
